SVM (Support Vector Machine)

Classification - Marge Maximale

🔍 Qu'est-ce que c'est ?

Les Machines à Vecteurs de Support cherchent à tracer la meilleure frontière possible (appelée hyperplan) pour séparer vos deux groupes : les personnes "Fit" et "Not Fit".

L'objectif est d'avoir la marge de sécurité la plus large possible entre cette frontière et les points de données les plus proches. Il utilise une astuce mathématique ("Kernel Trick") pour séparer des données complexes (non-linéaires).

⚙️ Caractéristiques

  • 📏 Marge Maximale : Excellente capacité de généralisation.
  • 🚀 Efficace : Très performant même avec peu de données mais beaucoup de dimensions.
  • 🧮 Noyaux (Kernels) : Capable de résoudre des problèmes très complexes.
  • Accuracy : 73%
  • Precision : 72%
  • F1_score : 0.64
  • Recall : 0.58
