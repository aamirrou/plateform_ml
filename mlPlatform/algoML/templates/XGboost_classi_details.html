XGBoost (Classification)

Boosting - État de l'art

🔍 Qu'est-ce que c'est ?

eXtreme Gradient Boosting est une méthode d'ensemble très avancée. Au lieu de créer des arbres indépendants (comme Random Forest), il crée des arbres séquentiellement.

Chaque nouvel arbre est construit spécifiquement pour corriger les erreurs commises par les arbres précédents. C'est comme une équipe où chaque nouvel expert se concentre uniquement sur les dossiers difficiles que les autres n'ont pas su traiter.

⚙️ Caractéristiques

  • 🚀 Performance : Souvent l'algorithme le plus précis sur les données tabulaires.
  • ⚡ Rapidité : Optimisé pour le calcul parallèle et l'utilisation mémoire.
  • 🔧 Régularisation : Intègre des mécanismes pour éviter le sur-apprentissage (Overfitting).
  • Accuracy : 73%
  • Precision : 72%
  • F1_score : 0.64
  • Recall : 0.58
