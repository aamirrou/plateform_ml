Régression Logistique

Classification - Probabiliste

🔍 Qu'est-ce que c'est ?

Malgré son nom (Régression), c'est un algorithme de Classification. Il ne prédit pas une valeur brute, mais la probabilité (entre 0 et 100%) que vous soyez "Fit".

[Image of sigmoid function graph]

Il utilise une courbe mathématique en forme de S appelée Fonction Sigmoïde. Si la probabilité calculée est supérieure à 50%, l'algorithme vous classe comme "Fit". Sinon, "Not Fit".

💡 L'Analogie Simple

C'est comme un examen avec une note de passage :
Le modèle calcule votre note sur 100 (la probabilité). Si vous avez plus de 50/100, vous réussissez l'examen (Classe 1 : Fit).

⚙️ Caractéristiques Clés

  • 📊 Sortie Probabiliste :
    Donne un indice de confiance précis (ex: "Fit à 87%").
  • 📏 Frontière Linéaire :
    Sépare les données par une ligne droite (ou un plan). Il fonctionne moins bien si les données sont mélangées de façon complexe.
  • ⚖️ Poids des variables :
    Les coefficients nous disent exactement quel facteur (ex: Poids vs Sommeil) a le plus d'impact sur la décision.
  • Accuracy : 73%
  • Precision : 72%
  • F1_score : 0.64
  • Recall : 0.58
