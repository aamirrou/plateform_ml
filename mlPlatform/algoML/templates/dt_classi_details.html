Arbre de Décision (Classification)

Classification - Hiérarchique

🔍 Qu'est-ce que c'est ?

Ce modèle fonctionne comme un jeu de "Qui est-ce ?". Il pose une série de questions binaires (Oui/Non) sur les caractéristiques de l'utilisateur.

Par exemple : "Fume-t-il ?" -> Si Non, "Fait-il du sport ?" -> Si Oui -> Classé FIT. Il divise la population en segments de plus en plus purs jusqu'à prendre une décision finale.

⚙️ Caractéristiques

  • 👁️ Visuel : Le modèle le plus facile à expliquer à un humain.
  • ⚡ Rapide : Prise de décision instantanée.
  • ✂️ Sensible : Un petit changement dans les données peut changer tout l'arbre.
  • Accuracy : 73%
  • Precision : 72%
  • F1_score : 0.64
  • Recall : 0.58
