Algorithme Random Forest (Classification)

Détails :

Le Random Forest (Forêt Aléatoire) est une méthode d'ensemble qui construit une multitude d'arbres de décision pendant l'entraînement. Pour la classification, la décision finale est prise par un vote majoritaire de tous les arbres. Cela corrige le défaut de sur-apprentissage des arbres simples.

Caractéristiques :

  • Robustesse : Très précis et stable.
  • Bagging : Utilise l'échantillonnage aléatoire.
  • Boîte noire : Moins interprétable qu'un arbre unique.

Random Forest (Classification)

Ensemble Learning - Classification

🔍 Qu'est-ce que c'est ?

Le Random Forest (Forêt Aléatoire) fonctionne comme une démocratie : il crée une multitude d'Arbres de Décision lors de l'entraînement.

Pour classer un profil (Fit ou Not Fit), chaque arbre donne son avis (vote). L'algorithme choisit ensuite la classe qui a reçu la majorité des votes. Cela permet de corriger les erreurs qu'un arbre seul pourrait faire.

⚙️ Caractéristiques

  • 🌲 La Force du Nombre : Plus robuste et précis qu'un arbre unique.
  • 🛡️ Anti-Surapprentissage : Réduit le risque d'apprendre les données "par cœur".
  • 📊 Importance des Variables : Permet de savoir quel critère (Poids, Age...) compte le plus.
